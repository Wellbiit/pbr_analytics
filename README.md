PBR Analytics

PBR Analytics - це проект, побудований на основі публічних даних з офіційного сайту Professional Bull Riders (pbr.com). Проект охоплює сезони 2020–2026 по 7 турах.

Проблема

Офіційний сайт PBR показує лише сезони і базові рейтинги. Немає можливості побачити кар'єрну динаміку райдера, його стабільність по роках і турах, або порівняти його з іншими на довгій дистанції. Цей проект заповнює цю прогалину.

Рішення

Повний аналітичний pipeline від збору даних до інтерактивного дашборду, який дозволяє фанатам, аналітикам і спонсорам оцінити результати райдерів у розрізі років, турів і кар'єри загалом.
